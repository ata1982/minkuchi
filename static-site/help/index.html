よくある質問

一般的な質問

Minkuchiは、企業やサービス、商品に関する実際の利用者の口コミ・レビューを共有するプラットフォームです。信頼できる情報を通じて、より良い選択をサポートします。

はい、Minkuchiは完全無料でご利用いただけます。レビューの閲覧、投稿、検索など、すべての基本機能を無料でお使いいただけます。

レビューの閲覧や検索は登録なしでもご利用いただけます。ただし、レビューの投稿やお気に入り機能を使うには、無料のアカウント登録が必要です。

レビューについて

アカウントにログイン後、「口コミを投稿」ボタンをクリックし、レビューしたいサービスを検索して選択してください。4つのステップに従って、評価とコメントを入力するだけで簡単に投稿できます。

はい、投稿後24時間以内であれば編集が可能です。削除はいつでも行うことができます。マイページの「投稿したレビュー」から操作してください。

レビューは登録されたユーザー名で表示されますが、プロフィール設定でニックネームを使用することができます。ただし、信頼性確保のため完全匿名での投稿はできません。

検索・閲覧

企業名、商品名、業界、地域などのキーワードで検索できます。フィルタ機能を使って評価順やレビュー数順で並び替えることも可能です。カテゴリページからも効率的に探せます。

ユーザー認証、重複投稿の防止、コミュニティによる相互評価システム、運営による定期的な監視により、信頼性の高いレビュー環境を維持しています。

技術的な問題

パスワードをお忘れの場合は「パスワードを忘れた方」リンクから再設定してください。それでも解決しない場合は、ブラウザのキャッシュをクリアするか、お問い合わせフォームからご連絡ください。

ブラウザの更新、キャッシュのクリア、別のブラウザでの確認をお試しください。モバイルアプリをご利用の場合は、アプリの再起動や最新バージョンへの更新をお試しください。
